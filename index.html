Pre-LLM Next-Word Prediction

Old n-gram models guess the next word from raw frequency alone. Write a passage below with one or more blanks marked <blank>, and watch a thousand guesses come out of the machine for each hole — then see how a real language model handles the same holes.

Set up the passage

Live options fetch a random slice of the real dataset from Hugging Face's public datasets-server API at run time (no key needed) and train on that instead — so results will vary run to run and depend on what the slice happens to contain. Google Books Ngrams isn't a choice here: it's published as raw n-gram frequency counts, not sentences, so there's no text in it for a sentence-level trainer to read.

Pick models to compare

Six stages of the same field, each on the same blank: unigram → bigram → trigram/4-gram (frequency tables, 1980s–2000s) → embeddings (words as vectors from co-occurrence statistics, the word2vec/GloVe-era idea, 2003–2013) → RNN (a running hidden state instead of a fixed window, Mikolov 2010) → LLM (a real pretrained transformer using full bidirectional context, 2018–present).

The LLM option downloads a ~65 MB model the first time (cached after that) and runs it fully in your browser — no server, no API key.

The Background corpus dropdown above can now train the n-gram, embeddings, and RNN models on a live random slice of a real dataset instead of just the small built-in one:

  • Wikitext-103 — the long-range-dependency Wikipedia language modeling benchmark n-gram and early neural LMs are usually evaluated on. Selectable above.
  • OpenWebText — the open-source recreation of the WebText corpus GPT-2 was trained on. Selectable above.
  • Wikipedia — the raw source most embedding models (word2vec, GloVe) and many LLM pretraining corpora draw from. Selectable above (via a Wikipedia snapshot, not the raw XML dumps directly).
  • Google Books Ngrams — n-gram counts (1–5 grams) over the full scanned Google Books corpus. Not selectable: it's published as frequency tables, not sentences, so there's no text in it for these models to train on.
